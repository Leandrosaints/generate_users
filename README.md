# Geração de usuarios aluno

## Descrição do Projeto

Este aplicativo desktop é desenvolvido em Python utilizando PyQt6 e Pandas. O objetivo do app é realizar o tratamento de dados de uma planilha, gerando um arquivo CSV que contém as informações necessárias para a criação de usuários e a execução de scripts PowerShell em um servidor.

## Tecnologias Utilizadas

- **Python**: Linguagem de programação principal.
- **PyQt6**: Framework para desenvolvimento de interfaces gráficas de usuário.
- **Pandas**: Biblioteca para manipulação e análise de dados.
- **PowerShell**: Utilizado para execução de scripts no servidor.

## Funcionalidades

- **Importação de Planilhas**: Permite ao usuário importar arquivos de planilhas (Excel).
- **Tratamento de Dados**: Processa os dados da planilha para extrair as informações necessárias.
- **Geração de CSV**: Cria um arquivo CSV com os dados tratados.
- **Execução de PowerShell**: Integração para executar scripts PowerShell diretamente do aplicativo.
